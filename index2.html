Solid Shapes

A solid shape is any solid or shape or object that has the 3 dimensions ...

length, breadth and height.

A Solid solid is nothing but a stack of 2d shapes. ​ For instance, if you stack a lot of squares linearly together, you get a cube and similarly if you stack a lot of rectangles together you get a cuboid.​

What If you stack the same rectangles in a circular manner like this? What will you end up with? Yes, you will end up with a Rectangle. ​

That is not the only way to create a cylinder right? If I stack a lot of circles linearly It will again result in a cylinder.
